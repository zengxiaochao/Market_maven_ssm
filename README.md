## 项目描述
### 这是大三下学期的JavaEE作业，JavaWeb，基于Maven的SSM框架做的一个商城，实现的功能有：
1. 浏览主页商品、查看商品详情、下单商品、查看我的订单、注册用户等功能。
2. 管理员查看商品列表、进行销量或售价或库存进行排序，进行商品上下架。
